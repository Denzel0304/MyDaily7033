# 의료 정보 (Medical Info)

응급실 / 약국 / 병의원 / 약 정보를 한 곳에서 빠르게 찾을 수 있는 모바일 웹 앱.

공공데이터포털 API를 활용한 단일 HTML 앱입니다.

## 🎯 주요 기능

- **🚑 응급**: 실시간 응급실 가용병상, 중증질환 수용 가능 여부, 외상센터
- **💊 약국**: 지금 운영 중인 약국, 24시간 약국, 야간 약국
- **🏥 병의원**: 진료과별 병원 검색
- **💉 약정보**: 이름으로 약 검색, 알약 모양으로 약 찾기

## 🔑 API 키 발급 (필수)

`index.html` 상단의 `API_KEYS` 객체에 공공데이터포털 인증키(Encoding 버전)를 넣어주세요.

### 활용신청 5개 (모두 자동승인, 무료)

1. **국립중앙의료원_전국 응급의료기관 정보 조회 서비스** (응급실/외상센터)
   - https://www.data.go.kr/data/15000563/openapi.do
   - → `API_KEYS.emergency`

2. **국립중앙의료원_전국 약국 정보 조회 서비스**
   - https://www.data.go.kr/data/15000576/openapi.do
   - → `API_KEYS.pharmacy`

3. **국립중앙의료원_전국 병·의원 찾기 서비스**
   - https://www.data.go.kr/data/15000736/openapi.do
   - → `API_KEYS.hospital`

4. **식품의약품안전처_의약품개요정보(e약은요)**
   - https://www.data.go.kr/data/15075057/openapi.do
   - → `API_KEYS.drugInfo`

5. **식품의약품안전처_의약품 낱알식별 정보**
   - https://www.data.go.kr/data/15057639/openapi.do
   - → `API_KEYS.drugPill`

### 인증키 확인 위치

공공데이터포털 → 마이페이지 → 데이터활용 → Open API → 인증키 확인  
**일반 인증키 (Encoding)** 값을 사용합니다.

> ⚠️ 활용신청 즉시 승인되지만, 실제 API 호출 가능까지 1~2시간 걸립니다.  
> "등록되지 않은 인증키" 에러가 뜨면 잠시 후 다시 시도하세요.

## 🔧 기술 스펙

- **단일 HTML**: 빌드 도구, 의존성 없음
- **모바일 최적화**: 600px 최대 너비, safe-area 대응 (노치/홈 인디케이터)
- **History API 기반 모달**: 뒤로가기로 모달 닫힘, 사이트 이탈 방지
- **GPS + 수동 위치 폴백**: 위치 권한 거부 시 시·도 선택
- **클라이언트 거리 계산**: Haversine 공식
- **카카오맵 길찾기 연동**: `https://map.kakao.com/link/to/`
- **localStorage**: 위치 설정 저장

## 🚧 알려진 한계

- 공공데이터 API는 시·도 단위 필터가 기본이라, GPS 모드라도 시·도를 추가로 선택하면 더 빠른 응답을 받을 수 있습니다.
- 응급실 가용병상 정보는 5분 단위로 갱신되며, 실제 상황과 차이가 있을 수 있습니다.
- 약국/병원 운영시간은 약국·병원이 직접 신고한 정보로, 변동 가능성이 있습니다. **방문 전 전화 확인 권장**.
